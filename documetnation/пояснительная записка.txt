Пояснительная записка
Название проекта: Telegram-бот для поиска мероприятий в разных городах
Цель проекта: Создать удобного Telegram-бота, который помогает пользователям находить актуальные события (концерты, выставки, театры, фестивали) в выбранном городе.
Основные функции:
	•	Выбор города (Москва, Санкт-Петербург, Казань)
	•	Выбор категории мероприятий
	•	Просмотр списка событий с краткой информацией
	•	Детальная информация о событии (дата, место, цена, описание)
	•	Ссылка на покупку билетов
Используемые технологии:
	•	Python 3.10+
	•	Библиотека python-telegram-bot (версия 20.x)
	•	API KudaGo для получения данных о событиях
	•	Логирование и обработка ошибок
Преимущества:
	•	Простота использования (интуитивно понятное меню)
	•	Быстрый доступ к актуальной информации
	•	Возможность вернуться назад или начать поиск заново
